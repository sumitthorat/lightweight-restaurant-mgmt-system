How to get the project running:
1. Clone the repository from https://github.com/sumitthorat/cs699-project
2. Navigate to the project directory and execute the following commands
3. Ensure all requirements are satisfied (will be installed if not satisfied): pip3 install -r Backend/API/requirements.txt
4. Run the API server: python3 Backend/API/main.py
5. Run the webapp: python3 Backend/Webapp/PROJECT/manage.py runserver
6. Install the APK Frontend/Android/app-debug.apk in the Android phone

How to access the webapp:
1. After setting up the project, scan the table specific QR code to open the webapp
2. Alternatively, type the url: http://localhost:8000/order/?table=1 in the web browser

How to access the Android app:
1. Copy the APK file to the mobile
2. Open the APK and let the mobile specific package manager install the APK (Needs "Install from unknown sources" option enabled in the settings)
